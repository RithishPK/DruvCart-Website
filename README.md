# DruvCart - E-Commerce Website

A full-stack e-commerce web application built with HTML, CSS (Tailwind), JavaScript, and Node.js + Express. Features product browsing, cart management, checkout, real-time order tracking, order cancellation, and a chatbot assistant.

## Features

- User authentication (Login / Sign Up)
- Product catalog with category filters and sorting
- Shopping cart with localStorage persistence
- Multi-step checkout (Shipping → Payment → Review)
- Real-time order status tracking (auto-updates every minute)
- Order cancellation
- Chatbot for product availability queries

## Tech Stack

| Layer    | Technology                        |
|----------|-----------------------------------|
| Frontend | HTML, CSS, Tailwind CSS, JavaScript |
| Backend  | Node.js, Express.js               |
| Storage  | localStorage (client-side)        |

## Prerequisites

- [Node.js](https://nodejs.org/) v14 or higher
- npm (comes with Node.js)

## Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/RithishPK/DruvCart-Website.git

# 2. Navigate into the project folder
cd DruvCart-Website

# 3. Install dependencies
npm install

# 4. Start the server
node server.js
```

Open your browser and go to: `http://localhost:3000`
